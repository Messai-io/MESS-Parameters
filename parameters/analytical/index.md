## Method Selection Guide

### By System Type

#### 🟢 Microbial Fuel Cells (MFCs)

**Essential Methods**:

- Cyclic Voltammetry (mechanism)
- EIS (resistance distribution)
- SEM (biofilm structure)
- FTIR (composition)

**Advanced Methods**:

- CLSM (3D biofilm)
- XPS (surface chemistry)
- Community Analysis (microbiology)

#### 🟡 Microbial Electrolysis Cells (MECs)

**Essential Methods**:

- Linear Sweep Voltammetry (overpotential)
- Chronoamperometry (stability)
- Gas Chromatography (products)
- SEM (catalyst)

**Advanced Methods**:

- In-situ Raman (intermediates)
- TEM (nanoparticles)
- Thermal Analysis (catalyst stability)

#### 🟣 Microbial Electrosynthesis (MES)

**Essential Methods**:

- Potentiostatic Testing (production)
- NMR (product identification)
- Mass Spectrometry (pathway)
- FTIR (monitoring)

**Advanced Methods**:

- Metabolomics (pathways)
- Isotope Labeling (mechanisms)
- SERS (surface species)

#### 🔴 Microbial Desalination Cells (MDCs)

**Essential Methods**:

- Conductivity Measurement (desalination)
- SEM (membrane fouling)
- Ion Chromatography (ion transport)
- Contact Angle (wettability)

**Advanced Methods**:

- XRD (scaling)
- AFM (membrane surface)
- Zeta Potential (fouling prediction)

### By Analysis Objective

#### Biofilm Analysis

1. **Structure**: SEM, CLSM, AFM
2. **Composition**: FTIR, Raman, XPS
3. **Activity**: CV, Fluorescence, Live-cell
4. **Community**: 16S, FISH, Metagenomics

#### Electrode Characterization

1. **Surface**: SEM, AFM, Contact Angle
2. **Electrochemistry**: CV, EIS, LSV
3. **Modification**: XPS, FTIR, Raman
4. **Stability**: Chronoamperometry, SEM

#### Product Analysis

1. **Identification**: NMR, MS, FTIR
2. **Quantification**: GC, HPLC, UV-Vis
3. **Purity**: MS, NMR, Chromatography
4. **Kinetics**: Real-time spectroscopy

#### Performance Optimization

1. **Bottleneck Identification**: EIS, CV
2. **Material Screening**: LSV, SEM
3. **Process Monitoring**: Chronoamperometry
4. **Degradation**: Multiple techniques

## Identification Criteria

A file was classified as an auto-generated stub if it:
1. Was under 2 KB in size
2. Contained the text "This parameter documentation was auto-generated"
3. Had only boilerplate sections with no specific data (no ranges, no citations, no equations)
4. Had a richer counterpart covering the same scientific concept

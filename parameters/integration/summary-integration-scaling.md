## Economic Scaling Models Summary

### Capital Cost Scaling

```
CAPEX(scale) = CAPEX(base) × (scale/base)^0.7
```

### Operating Cost Scaling

```
OPEX(scale) = OPEX(base) × (scale/base)^0.85
```

### Learning Curve Effects

```
Cost(n) = Cost(1) × n^(-log2(1-LR))
where LR = learning rate (typically 0.20)
```
